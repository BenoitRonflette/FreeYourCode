Global presentation

    FreeYourCode project is shared into 3 sub-projects:
  • FreeYourCode-Test-Generator which contains the core (transform input/output objects into use cases), the interceptors (currently, a Java Agent only), a web server (to control the test generation) and plugins (to adapt this tools to your context),
  • FreeYourCode-Test-Utils, a framework based on TestNG, which provides a structure to be able to execute the generated tests,
  • FreeYourCode-PrettyJson, based on JsonIo, which serializes objects from/to JSON but skipping the null and the default primitive values. It allows to add hooks before (de)serializing (for instance, to unproxy the Hibernate Objects).



Arrows show the dependecies between these 3 projects.


Make your own project

You can your own project enriching the initial one thanks to inheritance, factories or configuration (adding your own plugins).



Output example

  • (1) prepare the stubs(Mock Plugin)
  • (2) execute the tested method (Basic test generator)
  • (3) check the number of calls to the stubs (Mock Plugin)

@Test
public void testabstractCompute_0() throws Exception {
	//(1) Mock the stub methods
	Object[] abstractCompute_enter0 = new Object[]{JsonSerialisationUtils.deserialize("{\"@type\":\"double\",\"value\":7.0}")};
	Object[] abstractCompute_exit1 = new Object[]{JsonSerialisationUtils.deserialize("{\"@type\":\"double\",\"value\":7.0}")};
	Mockito.doAnswer(exitAnswer(abstractCompute_exit1)).when(genericClassImplStub).abstractCompute(Mockito.eq((Double)abstractCompute_enter0[0]));
		
	//(2) Call to tested method
	Object[] inputParams_enter = new Object[]{JsonSerialisationUtils.deserialize("{\"@type\":\"double\",\"value\":7.0}")};
	Object[] inputParams_exit = new Object[]{JsonSerialisationUtils.deserialize("{\"@type\":\"double\",\"value\":7.0}")};
	classCallingGenericClassImpl.abstractCompute((Double)inputParams_enter[0]);
	assertEquals(inputParams_exit, inputParams_enter);
		
	//(3)Check the number of calls to stub methods
	Mockito.verify(genericClassImplStub, Mockito.times(1)).abstractCompute((Double)Mockito.any());
}
