You can download the last Java Agent here. To be able to execute your generated tests, dependencies to FreeYourCode-Test-Utils and FreeYourCode-PrettyJson are required.


Maven dependencies: 
<dependency>
	<groupId>com.freeyourcode</groupId>
	<artifactId>test-utils</artifactId>
	<version>0.0.1-SNAPSHOT</version>
	<scope>test</scope>
</dependency>

<dependency>
	<groupId>com.freeyourcode</groupId>
	<artifactId>prettyjson</artifactId>
	<version>0.0.1-SNAPSHOT</version>
	<scope>test</scope>
</dependency>


All sources are available on GitHub.
